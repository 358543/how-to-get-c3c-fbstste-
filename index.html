Step-By-Step Guide

1. Install the Kiwi Browser

Install the Kiwi Browser from the Google Play Store and proceed to launch the application.

2. Enter the Download Link

Open the browser, navigate to the search bar, and enter the following link: Download Link.

3. Download the File

Click on the download option when prompted to download the file.

4. Access Extensions Menu

After downloading, tap the three dots in the top-right corner and select the Extensions option.

5. Enable Developer Mode

Enable Developer Mode and click "+" to add the extension using the downloaded file.

6. Install the Extension

Locate the downloaded zip file and click it to initiate the installation.

7. Activate the Extension

Once installed, activate the extension from the bottom-right corner.

8. Log In to Facebook

Upon activation, navigate to Facebook.com using the Kiwi Browser and log in to your account.

9. Locate C3C FBState Utility

After logging in, access the menu by clicking the three dots in the upper-right corner, scroll down, and locate the C3C FBState utility.

10. Copy Cookies

You'll be redirected to another page; locate the "Copy Clipboard" option and click it to copy your cookies for further use.
